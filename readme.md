# Краткий readme для запуска #

### Структура проекта ###

- `api` - папака с проектом на nodeJs + koa. Это бекенд проекта.
- `init-db` - содержит скрипт с запросом для создания таблицы в БД при инициализации проекта.
- `postman` - содержит коллекцию для Postman. В коллекции все CRUD запросы.
- `react` - папка с fronend частью проекта на React.
- `vue` - папака с fronend частью проекта на VueJS. Vue3 (CompositionAPI)


### Запуск проекта ###

> Поскольку весь проект контейнеризирован, то для запуска проекта необходим Docker. Для каждой сущности используется свой контейнер:
> - БД
> - API на NodeJs
> - FrontEnd на React
> - FrontEnd на VueJS

#### Запуск проекта ####

``` docker-compose up --build -d ```

#### Полная остановка проекта ####

``` docker-compose down -v ```
